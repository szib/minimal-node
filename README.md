# Minimal Node environment

Minimal Node environment with Babel, Jest, Eslint, Nodemon

## Installation

Clone the repo then run

```sh
  npm install
```

## Usage example

For starting nodemon on `src/app.js` run

```
  npm start
```

## Run tests

```
npm run test
// or
npm run test:watch
```

## Linting

```
npm run lint
// or
npm run lint:fix
```

## Build

```
// build to 'build' folder
npm run build
```

## Build binary

```
// build binary into 'bin' folder
npm run build:bin
```
